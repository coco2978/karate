Vocabulaire Karaté

Accueil Katas Vocabulaire

La liste ci-dessous n'est pas exhaustive mais rassemble les termes les plus utilisés.

Les commandements

  • Osu : Oui, compris, merci (dit lorsque le senseï a proposé un exercice)
  • Hajime : Commencer
  • Kiritsu : Debout, se redresser
  • Mawate : Tourner, pivoter
  • Mokusu : Méditation
  • Mokusu Yame : Fin de la méditation
  • Otagai Ni Rei : Salut entre élèves
  • Rei : Saluer
  • Sempai Ni Rei : Salut des anciens
  • Seïza : À genoux (se mettre)
  • Sensei Ni Rei : Salut au professeur
  • Shomen Ni Rei : Salut à « l’instance » supérieure au professeur
  • Yame : Fin, arrêtez
  • Yasume (Yassme) : Décontractez-vous, repos
  • Yoi : Soyez prêt, préparez-vous

Les termes de base

  • Chudan : Partie centrale (abdomen, poitrine)
  • Dachi : Position (des pieds et des jambes)
  • Dan : Niveau, degré (ceinture noire)
  • Do : Voie
  • Empi : Coude
  • Gedan : Partie inférieure (du corps)
  • Geri : Coup de pied
  • Gyaku : Contraire, opposé
  • Hidari : Gauche
  • Hiza : Genoux
  • Jodan : Partie supérieure (cou, tête)
  • Karategi : Tenue du karatéka
  • Kata : Ensemble de mouvements simulant un combat
  • Keage : Coup fouetté
  • Mae : Avant
  • Mawashi : Circulaire
  • Migi : Droite
  • Rei : Salut
  • Shuto : Tranchant de la main
  • Soto : Blocage extérieur
  • Tobi : Sauté
  • Tsuki : Coup de poing
  • Uchi : Blocage intérieur
  • Uke : Défense
  • Ura : Revers
  • Ushiro : Derrière
  • Yoko : Côté

Les techniques de poings

  • Gyaku-zuki : Coup de poing opposé à la jambe avant
  • Nukite : Pique avec les doigts
  • Kagi-zuki : Coup de poing latéral
  • Mawashi-zuki : Poing circulaire
  • Morote-zuki : Poing doublé
  • Oi-zuki : Poing en avançant

Les techniques de pieds

  • Kin-geri : Coup remontant (souvent aux parties)
  • Mae-geri : Avant
  • Mawashi-geri : Circulaire
  • Ushiro-geri : Arrière
  • Yoko-geri : Latéral
  • Yoko-tobi-geri : Latéral sauté

Les positions

  • Hachiji-dachi : Position normale
  • Heisoku-dachi : Pieds joints et parallèles
  • Kiba-dachi : Position du cavalier
  • Kokutsu-dachi : Position arrière
  • Musubi-dachi : Talons joints
  • Neko-ashi-dachi : Position du chat
  • Sanchin-dachi : Position du sablier
  • Shiko-dachi : Variante du cavalier pieds ouverts
  • Zenkutsu-dachi : Position avant

Les déplacements

  • De ashi / Ayumi ashi : En avançant
  • Hiki ashi : En reculant
  • Yori ashi : Pas glissé
  • Tsugi ashi : Pas chassé
  • Okuri ashi : Double pas
  • Mawari ashi : Rotation avant
  • Ushiro mawari ashi : Rotation arrière

Les défenses

  • Age-uke : Défense haute
  • Gedan-barai : Défense basse
  • Juji-uke : Blocage en croix
  • Kakiwake-uke : Défense de séparation
  • Morote-uke : Défense doublée
  • Shuto-uke : Défense du tranchant
  • Soto-uke : Défense extérieure
  • Teisho-uke : Défense paume vers l’avant
  • Uchi-uke : Défense intérieure
